# N Queens Problem Solver

## Overview

The N Queens puzzle is the challenge of placing N non-attacking queens on an NÃ—N chessboard. This program solves the N Queens problem for any board size `N` where `N` is greater than or equal to 4. It uses the backtracking algorithm to find and print all possible solutions where no two queens attack each other.

## Usage

### Prerequisites

- Python 3.x installed on your system.

### Running the Program

To solve the N Queens problem for a board of size `N`, run the script as follows:

```bash
./0-nqueens.py N
